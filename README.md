# Emo Hacker — Stranger Dracula

A dark VS Code theme blending Dracula, emo, and Stranger Things neon accents.

Preview locally

1. Open this folder in VS Code: `File -> Open Folder...` and select this extension folder.
2. Press `F5` to open an Extension Development Host with the theme loaded temporarily.
3. In the host window, open `Preferences: Color Theme` and select "Emo Hacker — Stranger Dracula".

Package & install

```bash
npm install -g vsce
vsce package
code --install-extension emo-hacker-stranger-0.0.1.vsix
```

Manual local install

Copy this folder to `~/.vscode/extensions/emo-hacker-stranger-0.0.1/` and restart VS Code.
